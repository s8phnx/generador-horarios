# Generador de horarios — Ingeniería + Formación Académica

Incluye:

- 76 ramos de Ingeniería (plan común + Industrial).
- 145 cursos de Formación Académica.
- 27 cursos deportivos.
- 118 cursos no deportivos.
- 37 cursos con modalidad B-learning.
- Selección automática del CFG que mejor encaja o elección de uno específico.
- Filtros por tipo y modalidad.
- Sin filtro de vacantes: esta herramienta es para planificar antes de la toma.
- Favoritos, profesores preferidos/a evitar, días libres y bloqueo de horarios.

## Subir a GitHub/Vercel

Reemplaza en tu repositorio:

- `index.html`
- `app.js`
- `styles.css`
- `data.js`
- `ramos.json`
